# Data Stream Accumulator

### 📝 Description
This application simulates a continuous data collection stream that logs and accumulates incoming integer inputs. It features a specific validation checkpoint where the loop execution continues dynamically until a designated termination trigger (the integer `5`) is parsed. The system separates the active entry metric from the core accumulator to avoid boundary pollution in the final sum.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `data-stream-accumulator.alg`.
3. Run the program.
