# Temperature Alert System

### 📝 Description
A weather reporting tool that captures a user-specified location and its local temperature in Celsius, processes the conversion formula, and outputs a formatted climate report in both Celsius and Fahrenheit.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `temperature-alert-system.alg`.
3. Run the program.
