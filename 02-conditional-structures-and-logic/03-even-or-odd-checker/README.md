# Even or Odd Checker

### 📝 Description
A fundamental mathematical logic utility that evaluates whether a user-provided integer is even or odd. The algorithm utilizes the modulo operator (`mod`) within a single conditional structure (`if/else`) to determine divisibility by two.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `even-or-odd-checker.alg`.
3. Run the program.
