# Commercial Property Estimator

### 📝 Description
This application simulates a real estate valuation dashboard for a modern 10-story high-rise building. It utilizes a structured multi-case match system (`switch-case`) to categorize real estate inventory based on floor elevation parameters. The engine maps distinct floor configurations (ground floor, intermediate floors, exclusive units, and the penthouse) and updates pricing matrices dynamically.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `commercial-property-estimator.alg`.
3. Run the program.
