# Factorial Math Processor

### 📝 Description
A structural mathematical algorithm engineered to compute the factorial product of any user-defined positive integer. The architecture implements a decrementing loop structure (`while counter > 1`), where the state tracking variable is continuously scaled down and multiplied against the running results block until the lowest evaluation boundary is reached.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `factorial-math-processor.alg`.
3. Run the program.
