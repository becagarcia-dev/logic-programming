# Multiplication Matrix Generator

### 📝 Description
An automated calculation matrix engine that constructs sequential arithmetic product tables. Upon receiving a base input multiplier, the system utilizes an active loop index tracker to generate a structured 10-row product matrix, outputting clean, formatted sequential data strings directly to the terminal interface.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `multiplication-matrix-generator.alg`.
3. Run the program.
