# Internet Cafe Billing System

### 📝 Description
A billing tool designed to manage customer usage in an internet cafe. The system captures the user's name, calculates total costs based on minutes used and an hourly rate, and generates a formatted receipt.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `lan-house.alg`.
3. Run the program.
