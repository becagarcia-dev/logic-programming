# Voter & Driver Status Validator

### 📝 Description
A demographic validation system that processes age inputs to determine legal status for voting and driving. It utilizes hierarchical nested conditional structures and relational boundaries to classify users into distinct legal categories based on localized age thresholds.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `voter-and-driver-status.alg`.
3. Run the program.
