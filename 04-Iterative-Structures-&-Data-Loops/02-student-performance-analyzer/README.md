# Student Performance Analyzer

### 📝 Description
An automated evaluation utility designed to process fixed-length multi-variable data blocks. The program accepts a string identifier (`studentName`) and runs a controlled conditional loop to collect exactly 4 numerical evaluation parameters (`grades`). It utilizes an internal index counter to dynamically guide the user prompt and computes the final arithmetic mean precisely.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `student-performance-analyzer.alg`.
3. Run the program.
