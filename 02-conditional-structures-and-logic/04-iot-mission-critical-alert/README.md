# IoT Mission-Critical Alert System

### 📝 Description
This challenge simulates a real-time monitoring system for the 'Yakutsk Autonomous Climate Hub & Edge Vault'. It incorporates telemetry data validation to intercept corrupt data entries. The core execution engine evaluates hardware vulnerabilities using complex **AND** / **OR** logic to toggle between full operational security, warning flags, and critical system alarms.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `iot-mission-critical-alert.alg`.
3. Run the program.
