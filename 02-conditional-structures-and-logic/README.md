# Module 02: Conditional Structures & Logic

### 📝 Description
This module focuses on mastering control flow structures and Boolean logic operators. The challenges explore decision-making paths using `if/else` (`se/entao/senao`) statements, nested conditions, and logical evaluations with **AND** (`e`) and **OR** (`ou`) operators to handle complex validation rules.

### 📊 Logic & Operators Reference
This mapping details how conditions and control flow operate in this section:

| VisuAlg Keyword | English Equivalent | Category | Description |
| :--- | :--- | :--- | :--- |
| `se ... entao` | `if ... then` | Control Flow | Initiates a conditional block based on a expression. |
| `senao` | `else` | Control Flow | Defines the alternative execution path if the condition is false. |
| `fimse` | `end if` | Control Flow | Finalizes the conditional structure scope. |
| `e` | `and` | Logical Operator | Returns `verdadeiro` ✅ only if **all** conditions are true. |
| `ou` | `or` | Logical Operator | Returns `verdadeiro` ✅ if at least **one** condition is true. |
| `nao` | `not` | Logical Operator | Inverts the boolean value (`verdadeiro` 🔄 `falso`). |
