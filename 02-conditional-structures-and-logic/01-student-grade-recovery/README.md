# Student Grade & Recovery System

### 📝 Description
An academic grading simulator that handles nested conditional structures (`if/else` branching). The system computes the initial arithmetic average of four grades. If the student falls below the passing threshold, it unlocks a secondary execution path to capture a recovery exam grade, recalculates the final score, and evaluates the ultimate approval status.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `student-grade-recovery.alg`.
3. Run the program.
