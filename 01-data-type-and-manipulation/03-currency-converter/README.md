# Currency Converter

### 📝 Description
A financial tool designed to standardize currency data. The system captures the user's name, accepts an amount in Brazilian Reais (BRL), and applies the current exchange rate to output a formatted conversion report in Euros (EUR).

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `currency-converter.alg`.
3. Run the program.
