# IoT Hub Command Center

### 📝 Description
An advanced multi-conditional dashboard built to control local telemetry. This application replaces verbose nested conditional loops with an optimized structural `switch-case` (`escolha/caso`) execution block. It simulates live hardware interactions, state manipulation, and memory calculation logic directly inside specific hardware routine paths.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `iot-command-center.alg`.
3. Run the program.
