# Smart Storage Monitor

### 📝 Description
An IoT-focused environmental monitoring simulation. The system captures storage temperature data, evaluates the threshold safety limits directly via boolean logic without control structures, and outputs the system's safety status.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `smart-storage-monitor.alg`.
3. Run the program.
