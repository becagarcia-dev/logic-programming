# IoT Smart Security Sensor

### 📝 Description
An internet-of-things (IoT) security simulation designed to handle device state changes. The system captures the number of motion-detected bodies in a room using an integer variable, evaluates a logical condition without control structures, and updates a boolean alarm state.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualgweb.com.br/).
2. Paste the code from `smart-security-sensor.alg`.
3. Run the program.
