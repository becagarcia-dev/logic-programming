# Input Boundary Validator

### 📝 Description
A robust data integrity gateway that enforces explicit range limits on incoming inputs. The application locks the execution flow inside a validation loop using a compound logical expression (`OR` evaluation), rejecting any integers outside the strict `1 to 10` scope. The loop acts as a security filter, granting script progression only upon receiving compliant criteria data.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `input-boundary-validator.alg`.
3. Run the program.
