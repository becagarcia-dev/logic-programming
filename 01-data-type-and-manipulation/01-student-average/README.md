# Student Average Calculator

### 📝 Description
An academic tool designed to calculate student performance. The system captures the student's name, accepts four separate grades, processes the arithmetic mean, and outputs a formatted final average report.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `student-average.alg`.
3. Run the program.
