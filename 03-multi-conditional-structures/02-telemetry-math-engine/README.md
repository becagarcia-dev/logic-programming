# Telemetry Math Processor

### 📝 Description
An algorithmic engine engineered to process and calibrate raw data metrics using multi-conditional arithmetic routing. The application hosts an interactive menu that dynamically accepts data blocks after an operation has been targeted. It includes structural mathematical exceptions, preventing system failure during core division execution routines when evaluating null constraints.

### 🛠️ Technologies
* VisuAlg (Pascal/Pseudocode)

### 🚀 How to Run
1. Open [VisuAlg Web](https://visualg.com.br/).
2. Paste the code from `telemetry-math-engine.alg`.
3. Run the program.
